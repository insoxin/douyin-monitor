# Douyin Monitor (抖音账号数据追踪面板)

基于 Cloudflare Workers 和 R2 对象存储构建的无服务器 (Serverless) 抖音账号数据追踪与可视化面板。无需购买服务器，利用 Cloudflare 免费额度即可实现零成本、免运维的全天候自动化监控。

## ✨ 核心特性

- 📊 **全维度数据追踪**：自动记录粉丝总数、关注数、作品数、获赞总数及喜欢数，并计算每日增减。
- 📱 **完美的 PWA 原生体验**：
  - 动态提取被监控用户的**抖音头像**作为 App 图标，**抖音昵称**作为应用名称。
  - 支持 iOS/Android/PC 将网页“添加到主屏幕”，享受沉浸式、无地址栏的全屏原生 App 体验。
- ⚡ **离线可用与秒开加载**：内置增强型 Service Worker，采用 Network First 缓存策略，支持弱网和断网环境下的页面秒开及历史数据查阅。
- 💾 **高可靠与存储优化**：
  - 采用 R2 分布式存储，实行“单日独立文件 + 全量视图动态构建”的读写分离策略，彻底避免并发写锁竞争。
  - 支持按需加载 (`?days=30`)，数据再多也能保持接口极速响应。
- 📈 **专业级数据可视化**：内置适配移动端手势交互的 Echarts 图表（防 Tooltip 边缘截断），直观呈现数据增长趋势。
- 🗂 **变更历史与数据导出**：自动记录该账号头像、昵称、个性签名的历史变更；支持一键将数据导出为 CSV 或 JSON 格式。
- 🔒 **隐私保护**：内置 Cookie 密码校验鉴权，防止数据面板被恶意窥探。
